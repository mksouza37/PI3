Bem vindo a API do Portal Pesque Bem. Nossa REST API oferece aos usuários a possibilidade da consultar via sistema a nossa base de dados com a situação de certificação ambiental das empresas do setor pesqueiro. A base pode ser consultada integralmente ou parcialmente, com o uso de de filtros pré-definidos. Caso essa seja a sua necessidade, basta seguir as orientações disponibilizadas nessa página.

Orientações para o uso da API do Portal Pesque Bem

A API oferece três alternativas de consulta: base integral, consulta de empresas por estado e consulta de empresa específica via CNPJ.
A tabela a seguir apresenta o formato da URL que a sua aplicação de seguir ao se comunicar com a API.


Tipo de Consulta Padrão de URL
Base Integral http://127.0.0.1:5000/api?SEL=todos
Empresas por Estado http://127.0.0.1:5000/api?EST=XX
Empresa por CNPJ http://127.0.0.1:5000/api?CNPJ=xx.xxx.xxxx/xx

A API responderá a requisição com um JSON array no seguinte formato: [{item},{item},{item},...].
